Browse the Pledger.io REST API below. Endpoints are grouped by tag; expand **Schemas** in the sidebar to inspect request and response entities.

Authentication uses a bearer JWT unless an operation is marked as public. The machine-readable specification is also available as [OpenAPI YAML](/openapi/openapi.bundle.yaml).

Transactions are the center of day-to-day bookkeeping in Pledger.io.
The transaction workflow is optimized for fast filtering and recurring review.

## Open the transaction pages

Use the left navigation and open **Transactions**.
Inside this section, use the local section navigation to switch between:

- **Income & expense transactions**
- **Transfers**
- **Import overview**
- **Start import**
- **Categories**

You can also use the global quick search (`Ctrl+K`) to jump directly to these pages.

{{< image src="img/how-to/transactions/transactions-overview.png" caption="Transactions overview with filters, presets, and saved views" >}}

## Work with filters

On the income/expense overview, the filter bar supports:

- Date and account filters
- Category and budget filters
- Search text and metadata filters
- Quick presets (for example all, income, expenses, uncategorized)

Active filters are shown as chips above the list.
Remove one chip to relax a single filter, or clear all chips to reset the view.

## Save recurring filter views

If you frequently use the same filter combinations, save them as a view preset.
Saved views let you switch between common review contexts quickly (for example monthly review, uncategorized cleanup, or budget follow-up).

Typical flow:

1. Set filters and presets to the desired view.
2. Save the view.
3. Re-apply or rename it later from the saved views controls.
4. Delete obsolete views when no longer needed.

## Review transfers separately

Transfers are managed on the dedicated **Transfers** page.
This keeps account-to-account movements separated from income/expense reporting and prevents accidental mixing in your spending analysis.

## Use the import workflow

The transactions section exposes two import destinations:

- **Import overview**: inspect previously imported files and continue existing imports.
- **Start import**: begin a new import run.

Use the overview when you need to audit what happened in a previous import, or continue processing later.
